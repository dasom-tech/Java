#프로그래밍이란?
- 프로그램을 만드는 것
- 프로그래밍 언어를 사용하여 프로그램을 제작한다.
- 프로그램을 만드는 사람을 프로그래머라고 부른다.

#프로그래밍 언어란?
- 컴퓨터는 0과 1밖에 알아듣지 못한다. 
- 사람이 알아들을 수 있는 언어와 컴퓨터가 알아들을 수 있는 언어의 중간지점이 프로그래밍 언어다.
- 프로그래머가 작성한 코드를 기계가 알아들을 수 있는 형태로 번역하는 것을 '컴파일'이라고 부른다.

#JDK (Java Development Kitm 자바 개발 도구)
- 자바로 개발을 하기 위한 여러가지 프로그램(컴파일러ㅡ 가바 실행 도구 등)들의 묶음
- 구글->JDK 검색 ->오라클 공식 홈페이지에서 다운로드(Windows X64, X86 맞춰서 설치)
- JDK버전은 8버전을 사용(대부분 7버전 사용)(전자정부프레임워크에서 사용하는 버전)

#JRE (Java Runtime Environment, 자바 실행 환경)
- 자바로 만든 프로그램을 실행할 때 필요한 프로그램
- 개발자는 재발 도중에 자기가 만든 프로그램을 계속 실행하면서 테스트하는것이 당연하기 때문에
  JDK를 설치하면 JRE도 함께 설치된다.

# IDE(Integrated Development Environment, 통합 개발 환경)
- 개발에 관련된 모든 작업을 하나의 프로그램으로 묶어놓은 것
- 개발을 하기 위해서는 "소스 코드 작성 -> 컴파일러로 컴파일 -> 실행환경으로 실행"
  하는 과정이 필연적으로 발생한다.
- 개발에 필요한 텍스트 에디터 / 컴파일러/ 실행환경을 하나의 프로그램으로 묶은 프로그램을
  통합 개발 환경이라고 부른다.

       ex) C언어의 IDE : Visual Studio, ...
           Python의 IDE : Visual Studio Code, Pycharm
           java의 IDE : IntelliJ, Eclipse
